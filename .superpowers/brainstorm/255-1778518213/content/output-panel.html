Output Panel: what does each mode render?

The right panel in the side-by-side layout. Select all modes that should be in Sprint 2. Unselected modes will show a "coming soon" placeholder.

T

Text — streaming tokens

Real-time token-by-token streaming via SSE. Text appears word-by-word in a cinematic monospace or display font. Copy, expand, and regenerate controls. Backed by HuggingFace Router (SmolLM3-3B or configurable model).

I

Image — progressive reveal

Single image generation via HF Inference API (FLUX.1-schnell). Loading shimmer while generating, then image fades in. Download, copy, save-to-gallery controls. Base64 → blob URL rendered in canvas.

V

Video — player with controls

Video generation via HF (Wan2.1-T2V-1.3B). Longer generation time (60–300s). Progress spinner with elapsed time. Output: HTML5 video player with autoplay, loop, download. Save to gallery.

A

Audio — waveform player

Audio generation via HF (musicgen-small). Animated waveform visualizer while generating. Output: audio player with waveform, play/pause, download. Save to gallery.
